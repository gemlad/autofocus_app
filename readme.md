Python script to use the late Mark Forster's autofocus (Final Version Perfected) method with a list of tasks from a Todoist filter.

http://markforster.squarespace.com/blog/2021/11/16/the-final-version-perfected-fvp-instructions-reposted.html
[todoist.com](https://www.todoist.com/)

The app will:
 - Retrieve a list of tasks from the Todoist API
 - Sequentially ask you to compare pairs of tasks and ask 'which would you want to do more than the other'
 - Provide the next action to do
 - Continue to ask which tasks you prefer, then provide the next action

Out of scope (for now):
 - writing anything to Todoist - tasks will have to be manually checked off in both systems
